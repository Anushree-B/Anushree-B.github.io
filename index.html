Political speech lie detection

I developed a speech recognition system using Python to transcribe spoken mathematical symbols, including Greek, Roman, and various others. Scraping over 200 symbols from the web, I trained a model to accurately convert speech to math symbols, challenging traditional input methods. Deploying the system on a local host using Streamlit, I explored new avenues in speech-to-text technology, enhancing accessibility to mathematical notation through innovative machine learning approaches.

Math Talk: A Speech to Symbol Approach

I developed a system using Python, NLTK, and neural networks to detect lies in political speeches. Starting with a dataset of 14,400 quotes scraped from PolitiFact.com, labeled for truthfulness, I analyzed various features including sentence length, part-of-speech distribution, sentiment, and vocabulary choice. Training the model improved accuracy to 46%, a 13% - 15% increase over the baseline. Additionally, I deployed the model on a cloud platform for scalability and accessibility.

Utilizing Python, OpenCV, and MediaPipe, I developed a real-time cursor control system based on hand gestures. This project not only improved my proficiency in computer vision and gesture recognition but also deepened my understanding of Python programming and its application in innovative HCI (Human-Computer Interaction) solutions.
